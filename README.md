# Nested Comments
A webapp made using React to demonstrate nested comments functionality.  
Link to live app server: https://noob-nested-comments.netlify.app/  
### Features:
- Add root and nested comments
- Edit Comment
- Delete Comment
- Upvote Comment
- Downvote Comment
